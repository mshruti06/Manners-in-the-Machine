## Project Overview
I caught myself saying "thank you" to a chatbot and turned that curiosity into rigorous HCI research. This project examines how courtesy in user prompts shapes AI responses, user self-perception, and cross-cultural interaction patterns.

**Key Findings:**
- Polite prompts generated 11% longer, more empathetic AI replies
- 68% of participants reported higher self-esteem after courteous exchanges
- Cultural differences: Indian users saw courtesy as "karma," Western users as "tech respect"
